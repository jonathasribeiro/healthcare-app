# Healthcare App

Reference architecture for a **healthcare patient scheduling** application using **React Native**, **Expo**, and **Firebase**.

## Planned stack

| Layer | Technology |
|-------|------------|
| Mobile | React Native + Expo |
| Auth & Data | Firebase |
| Language | TypeScript |

## Context

Inspired by enterprise healthcare platforms serving 500k+ users — patient booking, appointment management, and mobile-first UX for iOS and Android.

> This repository serves as a public reference. Production healthcare code is maintained in private repositories.
